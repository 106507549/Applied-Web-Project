# Applied-Web-Project
Applied Web Project Part 1 of 2 (Group Submission)

G05 – E‑Commerce & Digital Retail Platform
An online retail platform expanding its technical and user experience teams to enhance customer‑facing websites, product listings, and online application workflows.

https://swinburne.instructure.com/courses/71841/assignments/747254

Purpose and Scenario
You are required to build a static recruitment website using validated HTML5 and CSS3 only (no JavaScript or Bootstrap). The website must follow accessibility standards, use correct semantic structure, and be deployed using GitHub Pages.

Scenario: A company needs a website to advertise job vacancies. The website must include at least four pages: Home, Job Description, Apply, and About.. Based on the industry allocated to your group, design and develop a website for that company.
